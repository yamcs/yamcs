Time Ruler
==========

This band type shows absolute time markers. The scale takes different representations depending on the chart zoom level. The finest scale shows ``HH:mm:ss.SSS`` ticks every 10 milliseconds.


Configuration Options
---------------------

Timezone
    Select a IANA timezone, or ``UTC``.

    First select a region, next a sub-region, and for some sub-regions also the specific location.

    For example:

     * UTC
     * Europe/Brussels
     * America/Indiana/Indianapolis
