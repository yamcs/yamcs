# Security Policy

## Supported Versions

We support the **two most recent minor versions**. For example, if the current
version is 5.12.x, we provide security updates for 5.12.x and 5.11.x.

Please refer to our [Releases](https://github.com/yamcs/yamcs/releases) page
to identify the current latest versions.

## Reporting a Vulnerability

**Please do not use GitHub Issues to report security vulnerabilities.**

If you discover a potential security flaw, please report it privately via the
**Security and quality** tab of this repository and selecting
**"Report a vulnerability"**. This allows us to investigate and coordinate a
fix before the information is made public. 

We appreciate the community's help in keeping Yamcs secure. As we are a small
team, we follow the industry-standard **90-day disclosure window** to ensure
we have adequate time to develop and test patches.
